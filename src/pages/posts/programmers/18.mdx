# 아이스 아메리카노

## 문제 설명

머쓱이는 추운 날에도 아이스 아메리카노만 마십니다. 아이스 아메리카노는 한잔에 5,500원입니다.
머쓱이가 가지고 있는 돈 `money`가 매개변수로 주어질 때, 머쓱이가 최대로 마실 수 있는 아메리카노의 잔 수와 남는 돈을 순서대로 담은 배열을 return 하도록 solution 함수를 완성해보세요.

## 제한 사항

• 0 < `money` ≤ 1,000,000

## 입출력 예

| money | result |
| --- | --- |
| 5,500 | [1, 0] |
| 15,000 | [2, 4000] |

## 입출력 예 설명

입출력 예 #1

- 5,500원은 아이스 아메리카노 한 잔을 살 수 있고 잔돈은 0원입니다.

입출력 예 #2

- 15,000원은 아이스 아메리카노 두 잔을 살 수 있고 잔돈은 4,000원입니다.

## 문제풀이 (나)

```jsx
function solution(money) {
    const price = 5_500;

    const cup = Math.floor(money / price);
    const change = money - price * cup;

    return [cup, change];
}
```

시각적으로 한 눈에 소스가 들어오도록 구현했다.

## 문제풀이 (다른사람)

```jsx
function solution(money) {
    return [Math.floor(money / 5500), money % 5500];
}
```

나와 거의 비슷하게 풀었는데, 더 좋은 방법이다.

내가 남은 돈 계산하는 것 중에서 나머지 연산하면 쉽게 끝나는걸 놓쳤다.
