# 최댓값 만들기(1)

## 문제 설명

정수 배열 `numbers`가 매개변수로 주어집니다.
`numbers`의 원소 중 두 개를 곱해 만들 수 있는 최댓값을 return 하도록 solution 함수를 완성해주세요.

## 제한 사항

- 0 ≤ `numbers`의 원소 ≤ 10,000
- 2 ≤ `numbers`의 길이 ≤ 100

## 입출력 예

| numbers | result |
| --- | --- |
| [1, 2, 3, 4, 5] | 20 |
| [0, 31, 24, 10, 1, 9] | 744 |

## 입출력 예 설명

입출력 예 #1

- 두 수의 곱중 최댓값은 4 * 5 = 20 입니다.

입출력 예 #1

- 두 수의 곱중 최댓값은 31 * 24 = 744 입니다.

## 문제풀이 (나)

```jsx
function solution(numbers) {
    numbers = numbers.sort((a, b) => b-a);
    return numbers[0] * numbers[1];
}
```

배열 `numbers` 를 내림차순 정렬하여 큰 수 일수록 가장 첫번째부터 시작하도록 했다.

그리고 가장 첫번째 배열과 두번째 배열을 곱해서 리턴해주었다.

## 문제풀이 (다른사람)

```jsx
function solution(numbers) {
    numbers.sort((a,b)=>b-a);
    return numbers[0]*numbers[1];
}
```

나와 거의 비슷하게 풀었다.

그런데 내가 놓친점이, sort 는 원본 배열에도 영향을 준다.

나처럼 변수에 재할당해서 않아도 된다.
