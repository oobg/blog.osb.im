# 최댓값 만들기(2)

## 문제 설명

정수 배열 `numbers`가 매개변수로 주어집니다. `numbers`의 원소 중 두 개를 곱해 만들 수 있는 최댓값을 return하도록 solution 함수를 완성해주세요.

## 제한 사항

- 10,000 ≤ `numbers`의 원소 ≤ 10,000
- 2 ≤ `numbers` 의 길이 ≤ 100

## 입출력 예

| numbers | result |
| --- | --- |
| [1, 2, -3, 4, -5] | 15 |
| [0, -31, 24, 10, 1, 9] | 240 |
| [10, 20, 30, 5, 5, 20, 5] | 600 |

## 입출력 예 설명

입출력 예 #1

- 두 수의 곱중 최댓값은 -3 * -5 = 15 입니다.

입출력 예 #2

- 두 수의 곱중 최댓값은 10 * 24 = 240 입니다.

입출력 예 #3

- 두 수의 곱중 최댓값은 20 * 30 = 600 입니다.

## 문제풀이 (나)

```jsx
function solution(numbers) {
    let result = [];
    
    for (let i = 0; i < numbers.length; i++) {
        for (let j = 0; j < numbers.length; j++) {
            if (i === j) continue;
            result.push(numbers[i] * numbers[j]);
        }
    }

    result.sort((a, b) => b - a);
    return result[0];
}
```

결과 저장할 배열을 만들고, 이중 반복문을 사용해서 곱하기 결과를 모두 저장했다.

그리고 동일한 인덱스 값을 곱해주면 안되기 때문에, if문으로 동일 인덱스일 시 넘어가도록 했다.

마지막은 sort 를 사용해 가장 큰 값을 첫번째 배열로 오도록 했다.

## 문제풀이 (다른사람)

```jsx
function solution(numbers) {
    numbers.sort((a, b) => a - b);
    return Math.max(numbers[0]*numbers[1], numbers[numbers.length-1]*numbers[numbers.length-2]);
}
```

다른사람이 푼 방식인데 굉장히 흥미롭다.

먼저, 오름차순 정렬을 해준다.

이어서 배열의 가장 첫번째 두개 값을 곱한 것과, 배열의 마지막 두개 값을 곱한 것 중에, 큰 값을 가져오는 방식이다.
