# 자릿수 더하기

## 문제 설명

정수 `n`이 매개변수로 주어질 때 `n`의 각 자리 숫자의 합을 return 하도록 solution 함수를 완성해주세요

## 제한 사항

• 0 ≤ `n` ≤ 1,000,000

## 입출력 예

| n | result |
| --- | --- |
| 1234 | 10 |
| 930211 | 16 |

## 입출력 예 설명

입출력 예 #1

- 1 + 2 + 3 + 4 = 10을 return합니다.

입출력 예 #2

- 9 + 3 + 0 + 2 + 1 + 1 = 16을 return합니다.

## 문제풀이 (나)

```jsx
function solution(n) {
    const v = [...n.toString()].map(Number);
    return v.reduce((a, v) => a + v);
}
```

n 을 문자열로 바꾸고 구조분해했다.

사실상 n 이라는 문자열에 `.split(””)` 한 것과 동일하다.

그리고 map 연산자에 Number 파라미터를 줘서 모든 배열을 숫자 타입으로 변경해주었다.

마지막으로 reduce 연산자를 사용해서, 배열의 값을 더해주었다.

## 문제풀이 (다른사람)

```jsx
function solution(n) {
  return n
    .toString()
    .split("")
    .reduce((acc, cur) => acc + Number(cur), 0);
}
```
