# Node.js 설치 방법

## Node.js 설치

Node.js 를 설치하는 방법에 대해 알아보겠습니다.

Node.js는 다양한 방법으로 설치할 수 있으며, 이 글에서는 가장 일반적이고 편리한 방법을 소개합니다.<br />
모든 주요 플랫폼용 공식 패키지는 [Node.js 다운로드 페이지](https://nodejs.org/download/)에서 제공됩니다.

<Callout emoji="⚠️">
	최신 버전은 기능이 불안정하거나 일부 모듈이 작동하지 않을 수 있는 문제가 발생할 수 있습니다.<br />
	안정성이 보장되어 있는 이전 버전을 설치하시거나,
	비즈니스 환경이라면 오랜 기간 버전을 지원하는 LTS 버전을 설치하는 것을 권장합니다.
</Callout>

## 1. 패키지 매니저를 통한 설치

Node.js를 설치하는 가장 편리한 방법 중 하나는 패키지 매니저를 사용하는 것입니다.
운영 체제별로 다양한 패키지 매니저가 있습니다.
macOS, Linux, Windows용 패키지 매니저 목록은 Node.js 패키지 매니저 페이지에서 확인할 수 있습니다.

<Tabs items={["Windows", "macOS", "Linux"]}>
	<Tabs.Tab>
		`Windows`는 패키지 매니저로 `Chocolatey`를 사용합니다.<br />
		현재 작성일 기준(2024년 10월)으로 `Chocolatey` 에 올라온 Node.js 패키지의 최신 버전은 22.9.0 입니다.

		Windows PowerShell 또는 명령 프롬프트를 열고 다음 명령을 실행합니다.

		```bash
		choco install nodejs.install
		```

		하위 버전을 설치하려면 다음 명령을 실행합니다.
		예를 들어, 20 버전을 설치하려면 다음 명령을 실행합니다.

		```bash
		choco install nodejs --version=20.18.0
		```
	</Tabs.Tab>
	<Tabs.Tab>
		`macOS`는 패키지 매니저로 `Homebrew`를 사용합니다.<br />
		현재 작성일 기준(2024년 10월)으로 `Homebrew` 에 올라온 Node.js 패키지의 최신 버전은 22.9.0 입니다.

		Terminal 을 열고 다음 명령을 실행합니다.

		```bash
		brew install node
		```

		하위 버전을 설치하려면 다음 명령을 실행합니다.<br />
		예를 들어, 20 버전을 설치하려면 다음 명령을 실행합니다.

		```bash
		brew install node@20
		```
	</Tabs.Tab>
	<Tabs.Tab>
		`Linux`는 패키지 매니저로 `apt`를 사용합니다.

		Terminal을 열고 다음 명령을 실행합니다.

		```bash
		sudo apt install nodejs
		```
	</Tabs.Tab>
</Tabs>

<Callout emoji="⚠️">
	**패키지 매니저**의 설치 여부를 꼭 확인하시기 바랍니다.<br />
	각 운영 체제에 따라 패키지 매니저가 다르므로, 해당 운영 체제에 맞는 패키지 매니저를 사용해야 합니다.
</Callout>

## 2. NVM 을 통한 설치

Node.js를 실행하는 또 다른 방법이 있습니다.<br />
바로 `NVM`을 사용하는 것입니다.

`NVM`은 Node Version Manager의 약자로, Node.js의 버전을 관리하는 도구입니다.
NVM을 사용하면 여러 Node.js 버전을 쉽게 전환할 수 있고, 새로운 버전을 설치하거나 롤백할 수 있습니다.
이는 특히 이전 버전에서 코드를 테스트하거나 업데이트로 인한 문제를 해결할 때 유용합니다.

NVM에 대한 자세한 내용은 [NVM 공식 GitHub 페이지](https://github.com/nvm-sh/nvm)에서 확인할 수 있습니다.

<Tabs items={["Windows", "macOS", "Linux"]}>
	<Tabs.Tab>
		`Windows`는 패키지 매니저로 `Chocolatey`를 사용합니다.

		Windows PowerShell 또는 명령 프롬프트를 열고 다음 명령을 실행합니다.

		```bash
		choco install nvm.install
		```
	</Tabs.Tab>
	<Tabs.Tab>
		`macOS`는 패키지 매니저로 `Homebrew`를 사용합니다.

		Terminal 을 열고 다음 명령을 실행합니다.

		```bash
		brew install nvm
		```
	</Tabs.Tab>
	<Tabs.Tab>
		`Linux`는 패키지 매니저로 `apt`를 사용합니다.

		Terminal 에서 다음 명령을 실행합니다.

		```bash
		sudo apt install nvm
		```
	</Tabs.Tab>
</Tabs>

NVM 가 설치되었는지 확인하려면 다음 명령을 실행합니다.

```bash
nvm --version
```

위 명령을 실행하면 NVM의 버전이 출력됩니다.

이제, NVM 을 사용하여 Node.js 를 설치할 수 있습니다.<br />
우선 설치 가능한 node 목록을 확인해 보겠습니다.

```bash
nvm list-remote
```

원하는 버전을 선택하여 설치합니다.

```bash
nvm install 22.9.0
```

NVM 에 설치된 Node.js 버전을 현재 사용자 PC 에서 사용하도록 설정합니다.

```bash
nvm use 22.9.0
```

## 3. 설치 확인

Node.js가 정상적으로 설치되었는지 확인해 봅시다.

터미널 또는 명령 프롬프트에서 다음 명령을 실행합니다.

```bash
node -v
```

위 명령을 실행하면 Node.js의 버전이 출력됩니다.

<Callout type="info" emoji="🎉">
	**Node.js**가 성공적으로 설치되었습니다!<br />
	이제 Node.js를 사용하여 JavaScript 프로그램을 실행할 수 있습니다.
</Callout>
